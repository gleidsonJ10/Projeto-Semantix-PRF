# Projeto Semantix - Análise de Acidentes Rodoviários PRF 2023

## Problema
Os acidentes rodoviários representam um problema de segurança pública no Brasil, causando mortes, feridos e prejuízos econômicos.

## Coleta de Dados
Fonte: Polícia Rodoviária Federal (PRF)

Dados utilizados:
- Estado
- Total de acidentes
- Mortes
- Feridos graves

## Modelagem
Ferramentas utilizadas:
- Python
- Pandas
- Matplotlib
- Looker Studio

Etapas:
1. Limpeza dos dados
2. Análise exploratória (EDA)
3. Criação de métricas
4. Construção do dashboard

## Principais Insights
- Minas Gerais apresentou o maior número de acidentes.
- Minas Gerais também liderou em número de mortes.
- Os acidentes estão concentrados em determinados estados.

## Conclusão
A análise de dados permite identificar regiões críticas e apoiar decisões para redução de acidentes e melhoria da segurança nas rodovias.
